# 3 원정대 범인찾기 프로그램
3 원정대의 범인을 추적합니다.

## 기능
- 매 지구당 사망 카운트 수정

기본값은 3이지만, 마음에 드는 값으로 수정할 수 있습니다.  
단, 한 명당 1개의 데카를 보장하기 위해 최솟값은 3입니다.

- 플레이 중인 스트리머 입력란 제공

현재 플레이 중인 스트리머를 구분하기 위한 입력란을 제공합니다.

- 데카를 잘못 눌렀나요? 취소할 수 있습니다!

사망하다 버튼 아래의 "그건 틀렸어"를 누르면 데스 카운트를 1씩 줄일 수 있습니다.

- 진행 상황 추적

마인크래프트의 주요 진행 지점을 기준으로, 진행 상황을 추적할 수 있도록 했습니다.

[네더 진입], [블레이즈 막대 최초 획득], [엔드 진입], [엔드 해방]을 기준으로 버튼을 만들었습니다.

- 중단과 재개 지원

[파일 (F) > 저장 (S)]을 눌러 데이터를 저장할 수 있습니다.  
[파일 (F) > 불러오기 (L)]로 중단된 지점에서 계속할 수도 있습니다.
